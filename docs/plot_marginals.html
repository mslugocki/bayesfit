bayesfit.plot_marginals - Plot marginal distributions¶

bayesfit.plot_marginals(metrics)¶

Input Arguments¶

metricsdictionary

Output from bayesfit.fitmodel which contains important metrics from the fitted model to the data. (See desciption in bayesfit.fitmodel for full description of metrics output)

Output Variables¶

matplotlib.pyplot Object

Plot of marginal distributions for each parameter as computed from posterior surface.
